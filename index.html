Enchanter datapack

Minecrat Datapack to "craft" enchanting books.

View on github Download

Install

  • Download latest version
  • Place the zip file in the datapack folder of your world
  • Start crafting! :)

How to use

Unfortunatly it is not possible to craft enchanting books the regular way using a crafting table therefore the way to use this datapack is to ... just drop the ingredients on the ground together :). The game tick will detect and drop a book based on the ingredients dropped.
